# POS Product Description

An Odoo 17 module that allows the sale description of products to appear in the Point of Sale (POS) interface.

## Features

- Displays product descriptions in the POS
- Enhances product information visibility during sales transactions

## Installation

1. Add this module to your Odoo addons directory
2. Update the app list in Odoo
3. Install the module through the Apps menu

## Usage

Once installed, product descriptions will automatically appear in the POS interface when viewing product details.
